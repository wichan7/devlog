## Overview
As a follow-up to [MSA Design and Implementation - 01](https://wichan7.github.io/msa/msa-01/), I'm implementing a chatbot service.

## Current Configuration

Service Name|Description|GitHub
---:|:---|:---:
wic-api|Provides external integration functionality for the overall chatbot system|[Go to](https://github.com/wichan7/wic-api.git)
wic-app|Login, dashboard, and overall admin interface|[Go to](https://github.com/wichan7/wic-app.git)
wic-soe|Handles actual processing of the chatbot|-
wic-chatbot|Receives processed results from soe and provides web service|-
wic-mongo|Stores data that needs to be managed by the service|[Go to](https://github.com/wichan7/wic-mongo.git)
wic-redis|Used for managing volatile data like sessions|[Go to](https://github.com/wichan7/wic-redis.git)
wic-resources|Not a service, but registers yaml, images, etc.|[Go to](https://github.com/wichan7/wic-resources.git)
